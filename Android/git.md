###git

#####git命令：
git init
git add README.md  (或全部git add .)
git commit -m "first commit"
git remote add origin https://github.com/HuangFuFw/MarkDownFiles.git
git push -u origin master

再次提交部分到已经有的仓库：
git add 文件名
git commit -m "提交message"
git pull
git push
